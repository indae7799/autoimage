# PRD: 상세페이지 자동 생성 서비스 v1.0

## 1. 개요
브랜드/제품 이미지 분석(Gemini AI)을 통해 이커머스 상세페이지의 기획안과 카피라이팅을 자동 생성하고, 템플릿 기반으로 시각화 및 편집할 수 있는 도구.

---

## 2. 핵심 기능

### 2.1. 스마트 분석 및 분류 (Backend)
- **제품 분석**: 업로드된 이미지에서 브랜드명, 제품명, 핵심 성분 및 특징 자동 추출.
- **이미지 분류**: 이미지별 역할(메인, 제형, 성분, 사용법, 모델 등) 자동 지정.
- **컬러 추출**: 대표 이미지의 지배적인 색상을 분석하여 상세페이지 테마색으로 자동 매칭.

### 2.2. 컨텐츠 생성 (AI Planning)
- **섹션별 카피**: 후킹 헤드라인, 제품 특장점 설명, 사용법, 추천 대상 등 텍스트 자동 작성.
- **기획안 요약**: 전체 상세페이지의 흐름과 마케팅 포인트를 요약한 TOC 제공.
- **이미지 가이드**: 이미지가 없는 섹션을 위해 AI 이미지 생성용 프롬프트 제공.

### 2.3. 인터랙티브 편집 (Frontend)
- **실시간 수정**: 생성된 모든 텍스트의 폰트, 크기, 정렬, 색상 실시간 편집.
- **드래그 앤 드롭**: 섹션 재배치, 이미지 프레임 내 드래그 앤 드롭 교체.
- **디자인 조정**: 배경 이미지 변경 및 포인트 컬러 수정 기능.

### 2.4. 결과물 저장 및 활용
- **이미지 다운로드**: html2canvas 기반의 섹션별 이미지 결합 및 분할 저장.
- **배포 지원**: ZIP 파일 형태의 최종 결과물 제공.

---

## 3. 개발 로드맵 (Roadmap)

### Phase 1: 핵심 UI/UX 고도화 (현재)
- 카테고리별(화장품, 가전, 식품 등) 전문 템플릿 레이아웃 추가.
- 이미지 분석 결과와 템플릿 컬러 변수 간의 매핑 로직 정밀화.
- 편집기 내 이미지 삭제 및 개별 요소 정렬 기능 강화.

### Phase 2: 데이터 보강 및 외부 연동
- 구글 검색 API 연동을 통한 정확한 제품 정보(전성분, 인증 정보 등) 자동 보강.
- 리뷰 및 오픈마켓 데이터를 활용한 트렌디한 카피라이팅 알고리즘 개선.

### Phase 3: 플랫폼 최적화
- 네이버 스마트스토어, 쿠팡 등 주요 플랫폼 규격 자동 변환 및 등록 지원.

---

## 4. 기술 스택
- **프레임워크**: React (Frontend), FastAPI (Backend).
- **인공지능**: Google Gemini 1.5 Flash.
- **디자인/편집**: Tailwind CSS, @dnd-kit, html2canvas.

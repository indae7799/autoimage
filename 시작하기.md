# 🚀 서비스 실행 방법

## Windows에서 실행하기

### 방법 1: 명령 프롬프트(cmd) 사용 (가장 간단!)

1. **Windows 키 + R** 누르기
2. `cmd` 입력하고 Enter
3. 다음 명령어들을 순서대로 입력:

```cmd
cd C:\Users\jungindae\Downloads\상세페이지
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

4. **새로운 cmd 창**을 하나 더 열기 (Windows 키 + R → cmd)
5. 다음 명령어 입력:

```cmd
cd C:\Users\jungindae\Downloads\상세페이지
cd frontend
npm install
npm run dev
```

6. 브라우저에서 **http://localhost:5173** 접속!

---

### 방법 2: PowerShell 사용

1. **Windows 키 + X** 누르고 **Windows PowerShell** 선택
2. 다음 명령어 입력:

```powershell
cd C:\Users\jungindae\Downloads\상세페이지\backend
pip install -r requirements.txt
uvicorn main:app --reload
```

3. **새로운 PowerShell 창** 열기
4. 다음 명령어 입력:

```powershell
cd C:\Users\jungindae\Downloads\상세페이지\frontend
npm install
npm run dev
```

5. 브라우저에서 **http://localhost:5173** 접속!

---

### 방법 3: VSCode 터미널 사용 (가장 편함!)

1. VSCode에서 프로젝트 폴더 열기
2. **Ctrl + `** (백틱) 눌러서 터미널 열기
3. 터미널에서 **+** 버튼 눌러서 새 터미널 탭 추가 (2개 필요)
4. 첫 번째 터미널에서:

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

5. 두 번째 터미널에서:

```bash
cd frontend
npm install
npm run dev
```

6. 브라우저에서 **http://localhost:5173** 접속!

---

## ⚠️ 중요: 환경 변수 설정 필요!

실행하기 전에 `backend` 폴더에 `.env` 파일을 만들어야 합니다!

1. `backend` 폴더로 이동
2. `.env` 파일 생성 (메모장으로 만들면 됨)
3. 다음 내용 입력:

```
GEMINI_API_KEY=여기에_API_키_입력
```

**Gemini API 키 발급:**
- https://makersuite.google.com/app/apikey 접속
- Google 계정으로 로그인
- "Create API Key" 클릭
- 생성된 키 복사해서 `.env` 파일에 붙여넣기

## ❓ 문제 해결

### Python이 없다고 나오면:
- Python 설치: https://www.python.org/downloads/
- 설치 시 "Add Python to PATH" 체크!

### npm이 없다고 나오면:
- Node.js 설치: https://nodejs.org/
- 설치 후 터미널 재시작

### 포트가 이미 사용 중이라고 나오면:
- 다른 프로그램이 포트를 사용 중일 수 있습니다
- 해당 프로그램을 종료하거나 포트를 변경하세요

# 🚀 상세페이지 자동화 서비스

## 실행 방법

**`실행.bat` 파일을 더블클릭하세요!**

그러면 자동으로:
1. ✅ 백엔드 서버 실행
2. ✅ 프론트엔드 서버 실행
3. ✅ 브라우저 자동 오픈 (http://localhost:5173)

---

## 확인된 사항

- ✅ `.env` 파일이 `backend` 폴더에 존재합니다
- ✅ 환경 변수 설정 완료

---

## 접속 주소

- **웹사이트**: http://localhost:5173 (자동으로 열림)
- **API 문서**: http://localhost:8000/docs

---

## 서버 종료 방법

작업 표시줄에서 "백엔드"와 "프론트엔드" 명령 프롬프트 창을 닫으세요.

---

## 문제 해결

### 브라우저가 안 열리면
- 수동으로 http://localhost:5173 접속

### 서버가 실행되지 않으면
- Python 설치 확인: `python --version`
- Node.js 설치 확인: `node --version`
- 작업 표시줄의 명령 프롬프트 창에서 에러 메시지 확인

---

이제 `실행.bat`를 더블클릭하면 바로 사용할 수 있습니다! 🎉

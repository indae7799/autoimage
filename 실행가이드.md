# 상세페이지 자동화 서비스 실행 가이드

## 📍 확인 방법

서비스를 실행하면 다음 주소에서 확인할 수 있습니다:

- **프론트엔드 (웹사이트)**: http://localhost:5173
- **백엔드 API**: http://localhost:8000
- **API 문서 (Swagger)**: http://localhost:8000/docs

---

## 🚀 실행 단계

### 1단계: 환경 변수 설정

백엔드 디렉토리에 `.env` 파일이 필요합니다:

```bash
cd backend
```

`.env` 파일을 생성하고 다음 내용을 추가하세요:

```env
# 필수: Gemini API 키
GEMINI_API_KEY=your_gemini_api_key_here

# 선택사항: 구글 검색 기능 사용 시
GOOGLE_SEARCH_API_KEY=your_google_search_api_key
GOOGLE_SEARCH_ENGINE_ID=your_search_engine_id

# 또는 SerpAPI 사용 시
SERPAPI_KEY=your_serpapi_key
```

**Gemini API 키 발급 방법:**
1. https://makersuite.google.com/app/apikey 접속
2. Google 계정으로 로그인
3. "Create API Key" 클릭
4. 생성된 키를 복사하여 `.env` 파일에 붙여넣기

---

### 2단계: 백엔드 서버 실행

터미널 1번 창에서:

```bash
# 백엔드 디렉토리로 이동
cd backend

# Python 가상환경 생성 (선택사항, 권장)
python -m venv venv

# 가상환경 활성화
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

# 필요한 패키지 설치
pip install -r requirements.txt

# 서버 실행
uvicorn main:app --reload
```

**성공 메시지:**
```
INFO:     Uvicorn running on http://127.0.0.1:8000 (Press CTRL+C to quit)
INFO:     Started reloader process
INFO:     Started server process
INFO:     Waiting for application startup.
INFO:     Application startup complete.
```

이제 백엔드 서버가 실행 중입니다! ✅

---

### 3단계: 프론트엔드 실행

**새로운 터미널 창**을 열어서 (터미널 2번 창):

```bash
# 프론트엔드 디렉토리로 이동
cd frontend

# 필요한 패키지 설치 (처음 한 번만)
npm install

# 개발 서버 실행
npm run dev
```

**성공 메시지:**
```
  VITE v7.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

이제 프론트엔드가 실행 중입니다! ✅

---

### 4단계: 브라우저에서 확인

브라우저를 열고 다음 주소로 접속하세요:

**👉 http://localhost:5173**

웹사이트가 열리면:
1. 템플릿 선택 화면이 보입니다
2. 템플릿을 선택하고 "다음 단계로" 클릭
3. 이미지를 업로드
4. "상세페이지 생성하기" 클릭
5. 편집 화면에서 텍스트 수정 및 이미지 배치
6. "다운로드" 버튼으로 완성된 이미지 다운로드

---

## 🔧 문제 해결

### 백엔드가 실행되지 않는 경우

1. **Python이 설치되어 있는지 확인:**
   ```bash
   python --version
   ```
   Python 3.8 이상이 필요합니다.

2. **포트 8000이 이미 사용 중인 경우:**
   - 다른 프로그램이 포트 8000을 사용하고 있을 수 있습니다
   - `main.py` 파일에서 포트를 변경하거나 다른 프로그램을 종료하세요

3. **패키지 설치 오류:**
   ```bash
   pip install --upgrade pip
   pip install -r requirements.txt
   ```

### 프론트엔드가 실행되지 않는 경우

1. **Node.js가 설치되어 있는지 확인:**
   ```bash
   node --version
   npm --version
   ```
   Node.js 16 이상이 필요합니다.

2. **포트 5173이 이미 사용 중인 경우:**
   - 다른 프로그램이 포트 5173을 사용하고 있을 수 있습니다
   - Vite가 자동으로 다른 포트를 사용할 수 있습니다 (터미널 메시지 확인)

3. **패키지 설치 오류:**
   ```bash
   npm install --legacy-peer-deps
   ```

### API 연결 오류

1. **백엔드가 실행 중인지 확인:**
   - http://localhost:8000/docs 접속해서 API 문서가 보이는지 확인

2. **CORS 오류:**
   - 백엔드의 `main.py`에서 `origins` 리스트에 프론트엔드 주소가 포함되어 있는지 확인

3. **API 키 오류:**
   - `.env` 파일에 `GEMINI_API_KEY`가 올바르게 설정되어 있는지 확인
   - 백엔드 서버를 재시작하세요 (환경 변수 변경 후)

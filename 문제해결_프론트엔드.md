# 프론트엔드 서버 문제 해결

## 문제: 프론트엔드 터미널에 메시지가 안 나옴

### 원인 확인

프론트엔드 서버 창에서 어떤 메시지가 나오는지 확인하세요:

1. **"프론트엔드 서버" 창을 확인하세요**
2. 에러 메시지가 있는지 확인
3. 어떤 단계에서 멈췄는지 확인

---

## 해결 방법

### 방법 1: 수동으로 실행 (가장 확실함)

**`프론트엔드_수동실행.bat` 파일을 더블클릭하세요!**

또는 명령 프롬프트에서:

```cmd
cd C:\Users\jungindae\Downloads\상세페이지\frontend
npm install
npm run dev
```

이렇게 하면 정확히 어떤 에러가 발생하는지 볼 수 있습니다.

---

### 방법 2: 일반적인 문제 해결

#### 1. "npm이 인식되지 않습니다"
- Node.js 설치 필요: https://nodejs.org/
- 설치 후 명령 프롬프트 재시작

#### 2. "포트가 이미 사용 중입니다"
- 다른 프로그램이 포트 5173을 사용 중
- 작업 관리자에서 Node 프로세스 종료
- 또는 컴퓨터 재시작

#### 3. "모듈을 찾을 수 없습니다"
- 패키지 설치 필요:
  ```cmd
  cd frontend
  npm install
  ```

#### 4. "Cannot find module" 에러
- node_modules 폴더 삭제 후 재설치:
  ```cmd
  cd frontend
  rmdir /s /q node_modules
  npm install
  ```

---

### 방법 3: 포트 확인

포트가 5173이 맞는지 확인:

1. `frontend/vite.config.js` 파일 확인
2. `port: 5173`으로 설정되어 있는지 확인
3. 만약 3000이면 5173으로 변경

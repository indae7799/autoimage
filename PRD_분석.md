# 상세페이지 자동화 서비스 - 요구사항 분석 문서

## 📋 프로젝트 개요
쿠팡/네이버 등 이커머스 플랫폼에 제품을 등록하고 상세페이지를 자동으로 생성하는 서비스

---

## 🎯 핵심 요구사항

### 1. 템플릿 기반 시스템
**현재 상태**: ❌ 미구현
- **필요 기능**:
  - 미리 정의된 상세페이지 템플릿 저장소
  - 템플릿 선택 UI (템플릿 갤러리)
  - 템플릿 구조 정의 (섹션 타입, 레이아웃, 색상 변수)

**구현 방향**:
- 템플릿 JSON 스키마 정의
- 템플릿 저장소 (로컬/DB)
- 템플릿 선택 컴포넌트 추가

---

### 2. 이미지 색상 분석 및 자동 색상 매칭
**현재 상태**: ⚠️ 부분 구현 (색상 팔레트 추출만 있음)
- **필요 기능**:
  - 업로드된 제품 이미지에서 주요 색상 추출
  - 템플릿의 포인트 색상 변수 자동 변경
  - 메인제목/부제목 색상과 본문 색상 구분 적용

**현재 구현**:
- `backend/main.py`의 `/api/analyze` 엔드포인트에서 `colorPalette` 추출
- `frontend/src/App.jsx`에서 `theme` 객체로 색상 전달

**개선 필요**:
- 이미지에서 더 정확한 색상 추출 알고리즘
- 템플릿 색상 변수 매핑 시스템
- 제목/본문 색상 자동 대비 계산

---

### 3. 정보 추출 및 생성
**현재 상태**: ✅ 부분 구현 (이미지 분석만 있음)
- **필요 기능**:
  - 이미지에서 제품 정보 추출 (브랜드, 제품명, 성분 등)
  - **구글 검색 연동**으로 제품 정보 보강
  - 헤더, 후킹문구, 핵심성분, 설명, 사용법 자동 생성
  - 템플릿 지정 위치에 자동 배치

**현재 구현**:
- Gemini AI를 통한 이미지 분석 (`/api/analyze`)
- 상세페이지 기획안 생성 (`/api/plan`)

**개선 필요**:
- 구글 검색 API 연동 (Custom Search API 또는 SerpAPI)
- 제품명/브랜드명 기반 정보 검색
- 검색 결과를 AI 프롬프트에 통합

---

### 4. 텍스트 편집 기능
**현재 상태**: ✅ 구현됨
- **구현된 기능**:
  - `SectionBlock.jsx`의 `EditableText` 컴포넌트
  - 텍스트 필드 클릭 시 편집 가능
  - 폰트 크기, 굵기, 정렬, 색상 변경 가능

**개선 필요**:
- 더 직관적인 편집 UI
- 실시간 미리보기 개선

---

### 5. 드래그 앤 드롭 이미지 편집
**현재 상태**: ✅ 부분 구현
- **구현된 기능**:
  - `DragDropEditor.jsx`에서 섹션 재정렬
  - 이미지를 섹션에 드롭하여 배치
  - 배경 이미지 설정

**필요 기능**:
- ✅ 섹션별 이미지 프레임에 드래그 앤 드롭 (구현됨)
- ✅ 배경 이미지 삽입/제거 (구현됨)
- ⚠️ 이미지 제거 버튼 추가 필요
- ⚠️ 이미지 크기 조정 기능 (선택사항)

**개선 필요**:
- 이미지 제거 버튼 UI 추가
- 이미지 크기/위치 조정 기능 (선택사항)

---

### 6. 최종 이미지 다운로드
**현재 상태**: ✅ 구현됨
- **구현된 기능**:
  - `DownloadControl.jsx`에서 html2canvas로 섹션별 이미지 생성
  - ZIP 파일로 다운로드
  - 폴더명 설정

**개선 필요**:
- 이미지 품질 설정 옵션
- 단일 이미지로 합치기 옵션 (선택사항)

---

## 📊 현재 프로젝트 구조

### 백엔드 (`backend/`)
- **기술 스택**: FastAPI, Gemini AI
- **주요 엔드포인트**:
  - `POST /api/analyze`: 이미지 분석 및 색상 추출
  - `POST /api/plan`: 상세페이지 기획안 생성

### 프론트엔드 (`frontend/`)
- **기술 스택**: React, Vite, Tailwind CSS, @dnd-kit
- **주요 컴포넌트**:
  - `App.jsx`: 메인 플로우 관리
  - `DragDropEditor.jsx`: 드래그 앤 드롭 편집기
  - `SectionBlock.jsx`: 섹션 렌더링 및 편집
  - `DownloadControl.jsx`: 이미지 다운로드
  - `Sidebar.jsx`: 이미지 목록 및 프롬프트

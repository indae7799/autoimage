# 문제 해결 가이드

## 실행하기.bat가 안 될 때

### 어떤 에러가 나오나요?

1. **"Python이 설치되어 있지 않습니다"**
   - 해결: Python 설치 필요
   - https://www.python.org/downloads/
   - 설치 시 "Add Python to PATH" 체크 필수!

2. **"Node.js가 설치되어 있지 않습니다"**
   - 해결: Node.js 설치 필요
   - https://nodejs.org/

3. **"포트가 이미 사용 중입니다"**
   - 해결: 다른 프로그램이 포트를 사용 중
   - 작업 관리자에서 Python이나 Node 프로세스 종료
   - 또는 컴퓨터 재시작

4. **"모듈을 찾을 수 없습니다"**
   - 해결: 패키지 설치 필요
   - 수동으로 설치:
     ```cmd
     cd backend
     pip install -r requirements.txt
     
     cd ..\frontend
     npm install
     ```

5. **브라우저가 안 열림**
   - 해결: 수동으로 접속
   - http://localhost:5173 직접 입력

## 체크리스트

실행 전에 확인:

- [ ] Python 설치됨 (`python --version` 확인)
- [ ] Node.js 설치됨 (`node --version` 확인)
- [ ] `backend\.env` 파일 존재
- [ ] `backend\.env`에 `GEMINI_API_KEY` 설정됨
- [ ] 포트 8000, 5173 사용 가능

## 디버깅 모드

문제를 정확히 파악하려면:
